# CV Modular en React – Carlos Herrera

Proyecto desarrollado como parte de la evidencia **GA1-220501096-03-AA1-EV05 – Construcción de un CV modular en React**, donde se implementa una hoja de vida completamente modularizada usando componentes funcionales y anidados.

## Descripción del Proyecto

Este proyecto consiste en una **hoja de vida (CV)** desarrollada con **React**, donde cada sección (información personal, educación, experiencia, habilidades, etc.) está construida como un componente independiente.  
El objetivo es aplicar buenas prácticas de **modularización, reutilización y control de versiones con Git.**
